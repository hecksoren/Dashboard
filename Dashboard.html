        ☁️ Cloud Sync

        Verbinde dein Dashboard mit Firebase um es von jedem Gerät zu nutzen. Einmalige Einrichtung (~5 Min), danach alles automatisch.

        1
        Firebase-Projekt erstellen
        Öffne console.firebase.google.com → Projekt erstellen → Namen eingeben (z.B. „mein-dashboard") → Weiter bis fertig.
        2
        Web-App hinzufügen & Config kopieren
        Im Projekt: Projekteinstellungen → Web-App hinzufügen (</>) → App-Nickname eingeben → Registrieren. Den gesamten firebaseConfig = { ... } Block als JSON hier einfügen:
        3
        Firestore & Google Auth aktivieren
        Im Firebase-Menü: Firestore Database → Erstellen (Testmodus reicht). Dann: Authentication → Anmeldeanbieter → Google → Aktivieren.
        ✓ Verbunden — Daten werden automatisch synchronisiert
